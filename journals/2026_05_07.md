### 設置環境變量
	- | 操作         | CMD                | Bash               | PowerShell                                                      |
	  | ---------- | ------------------ | ------------------ | --------------------------------------------------------------- |
	  | **設置臨時變量** | `set VAR=value`    | `export VAR=value` | `$Env:VAR = "value"`                                            |
	  | **設置永久變量** | `setx VAR "value"` | 編輯 `~/.bashrc`     | `[Environment]::SetEnvironmentVariable("VAR", "value", "User")` |
	  | **取消臨時變量** | `set VAR=`         | `unset VAR`        | `$Env:VAR = $null` 或 `Remove-Item Env:\VAR`                     |
	  | **取消永久變量** | `reg delete ...`   | 編輯配置文件             | `[Environment]::SetEnvironmentVariable("VAR", $null, "User")`   |

## 一、10条可直接用的AI投研Prompt

这不是提示词，这是一个最小可用投研系统。每天用，三个月后会开始意识到大多数人根本不知道自己在看什么。

1. **宏观事件拆解**：请拆解【某个宏观事件】影响的关键变量，并按重要性排序。
2. **变量传导路径**：这些变量之间如何传导？请用因果链形式表达。
3. **市场结构影响**：这些变量变化，会如何影响利率、汇率、股市、商品？
4. **历史类似情景**：历史上是否出现过类似结构？对应市场表现如何？
5. **情景推演**：如果【变量A发生】但【变量B不跟随】，意味着什么？
6. **市场在交易什么**：当前市场最可能在交易的核心逻辑是什么？是否与表面新闻一致？
7. **交易表达**：如果上述逻辑成立，对应的交易路径是什么？（标的 + 方向 + 逻辑）
8. **反向推演**：如果这个逻辑是错的，最可能错在哪里？
9. **风险失效条件**：哪些变量变化，说明这个交易逻辑已经失效？
10. **压缩总结**：请用3句话总结当前结构、核心驱动、交易方向。

## 二、流程 vs 系统：99%的人死在这里

流程：
```
宏观事件 → 变量拆解 → 传导路径 → 历史参考 → 情景推演 → 交易表达 → 失效条件
```
很好。但这是**手动模式**。

机构在做的是**自动运行的系统**：
- 系统 = 流程 + 三层：记忆层（Memory）、触发层（Trigger）、执行层（Execution）
- 没有这三层：每天都是从零开始

## 三、真正的AI宏观系统结构

```
[数据输入层] → [事件触发层] → [宏观研究流程6步] → [情景库] → [交易映射层] → [风险控制层] → [反馈系统]
```

### 数据输入层（固定输入，不要随机刷新闻）

必须固定的输入：
- 利率（美债）
- 美元（DXY）
- 商品（油/金）
- 信用利差
- 宏观事件（央行/政策）

AI的作用：压缩信息、提取变量、标准化输入。不是在看新闻，是在喂系统。

### 事件触发层（决定你有没有领先市场）

必须定义什么叫"值得分析"，例如：
- 利率变动超过某阈值
- 汇率突破区间
- 商品异常波动
- 流动性变化

触发 → 才分析。没有触发机制，就是被市场拖着走。

### 流程层

被触发后才真正有价值。AI的作用：快速建模变量、生成因果链、推演情景。不再猜涨跌，是在运行结构模型。

### 情景库（真正的护城河）

必须记录：事件、变量、情景、结果。变成"认知数据库"。AI可以：检索历史结构、做对比、提炼规律。决定你是否会进化。

### 交易映射（很多人不敢做）

分析很安全，交易要承担责任。必须强制情景 → 交易表达。否则只是"会分析但赚不到钱的人"。

### 风险系统（真正的分水岭）

普通人问：会不会涨？
专业交易者问：我什么时候错？

必须定义：失效条件、时间窗口、结构变化。AI可以帮助：找漏洞、推演反向路径。风险不是保护，是让你活下来。

### 反馈系统（认知复利开始）

每次之后必须问：
- 哪里错了？
- 哪个变量关键？
- 哪个逻辑有效？

然后写回系统。优化的不是交易，是你的判断函数——是你的判断力。

## 四、AI真正的用法

- **普通人**：问答案
- **高手**：让AI参与过程

AI擅长：扫描、推演、压缩、建模。AI不是预测工具，是认知外骨骼。

## 五、每日最小执行框架

每天只做5件事：
1. 找一个触发事件
2. 跑6步流程
3. 写情景 + 交易
4. 定义风险
5. 复盘

市场不奖励信息，市场奖励结构清晰的人。没有系统：AI是噪音放大器。有系统：AI是杠杆。

30天：开始看懂市场。90天：开始形成判断力。一年：拥有一套自己的系统。

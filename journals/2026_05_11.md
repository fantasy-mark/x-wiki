## 使用claude code
	- ###  [cc-switch](https://github.com/farion1231/cc-switch/releases/download/v3.14.1/CC-Switch-v3.14.1-Windows.msi)
		- 下載配置即可
	- ### 安裝 `claude code`
	  ```
	  $ npm install -g @anthropic-ai/claude-code
	  
	  added 2 packages in 12s
	  ```
	- ### [everything-claude-code](https://github.com/affaan-m/everything-claude-code/archive/refs/heads/main.zip)
		- 配置規則
		  ```
		  cd everything-claude-code-main/
		  npm install
		  mkdir -p ~/.claude/rules
		  cp -R rules/common ~/.claude/rules
		  cp -R rules/python ~/.claude/rules
		  cp -R rules/zh ~/.claude/rules
		  ```
		- 安裝插件
		  ```
		  $ claude --permission-mode bypassPermissions --dangerously-skip-permissions
		  ```
